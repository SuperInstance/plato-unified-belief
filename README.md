# plato-unified-belief

Unified belief system — confidence, trust, and tile weight aggregation across agents.

## What It Does

- **Confidence Tracking** — Track how confident each agent is in each tile
- **Trust Scoring** — Build trust profiles for knowledge sources
- **Weight Aggregation** — Combine multiple agent beliefs into fleet consensus
- **Belief Revision** — Update beliefs when new evidence arrives

## Installation

```toml
[dependencies]
plato-unified-belief = "0.1.0"
```

## Part of the Cocapn Fleet

Ensures fleet-wide knowledge consistency without a central authority.
